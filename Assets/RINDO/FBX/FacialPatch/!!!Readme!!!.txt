Unityプロジェクト内で「start.bat」を実行しないでください。
右クリックし、Show in Explorerを選択してWindows Explorerで「start.bat」を実行します。


Unity 프로젝트 상에서 start.bat 를 실행하지 마세요.
오른클릭 후 Show in Explorer를 선택하여 윈도우 탐색기 상에서 실행해 주세요.


Do not run start.bat from within your Unity project.
Run 'start.bat' it in Windows Explorer by right-click and selecting Show in Explorer.
